


===========================================
 IntelliJ IDEA 유용한 단축키 정리
===========================================

[기본 조작 및 탐색]

- 최근 파일 보기: Ctrl + E
- 최근 편집한 위치로 이동: Ctrl + Shift + Backspace
- 클래스 찾기: Ctrl + N
- 파일 찾기: Ctrl + Shift + N
- 심볼 찾기: Ctrl + Alt + Shift + N
- 전체 검색: Shift + Shift
- 현재 파일 구조 보기: Ctrl + F12
- 프로젝트 창 포커스: Alt + 1
- 에디터로 포커스 이동: Esc
- 이전/다음 위치로 이동: Ctrl + Alt + Left/Right
- 이전/다음 탭으로 이동: Ctrl + Tab / Ctrl + Shift + Tab

[코드 편집 및 작성]

- 기본 코드 완성: Ctrl + Space
- 스마트 코드 완성: Ctrl + Shift + Space
- 현재 문장 완성: Ctrl + Shift + Enter
- 빠른 수정 및 의도 액션: Alt + Enter
- 현재 줄 복제: Ctrl + D
- 현재 줄 삭제: Ctrl + Y
- 현재 줄 또는 선택 영역 주석 처리: Ctrl + /
- 선택 영역 블록 주석 처리: Ctrl + Shift + /
- 코드 포맷팅: Ctrl + Alt + L
- 들여쓰기 자동 정렬: Ctrl + Alt + I
- 선택 영역 확장/축소: Ctrl + W / Ctrl + Shift + W

[코드 탐색 및 분석]

- 선언으로 이동: Ctrl + B
- 구현으로 이동: Ctrl + Alt + B
- 상위 메서드/클래스로 이동: Ctrl + U
- 사용처 찾기: Alt + F7
- 빠른 사용처 보기: Ctrl + Alt + F7
- 메서드 파라미터 정보 보기: Ctrl + P
- 문서 보기: Ctrl + Q
- 타입 계층 보기: Ctrl + H
- 호출 계층 보기: Ctrl + Alt + H

[리팩토링]

- 리팩토링 메뉴 호출: Ctrl + Alt + Shift + T
- 변수 추출: Ctrl + Alt + V
- 메서드 추출: Ctrl + Alt + M
- 필드 추출: Ctrl + Alt + F
- 상수 추출: Ctrl + Alt + C
- 파라미터 추출: Ctrl + Alt + P
- 이름 변경: Shift + F6
- 시그니처 변경: Ctrl + F6
- 코드 감싸기 (if, try 등): Ctrl + Alt + T

[검색 및 치환]

- 현재 파일에서 찾기: Ctrl + F
- 현재 파일에서 바꾸기: Ctrl + R
- 전체 프로젝트에서 찾기: Ctrl + Shift + F
- 전체 프로젝트에서 바꾸기: Ctrl + Shift + R
- 다음/이전 검색 결과로 이동: F3 / Shift + F3

[디버깅 및 실행]

- 실행: Shift + F10
- 디버깅 시작: Shift + F9
- 디버깅 중 중단: Ctrl + F2
- 다음 줄로 이동 (Step Over): F8
- 메서드 내부로 이동 (Step Into): F7
- 스마트 스텝 인: Shift + F7
- 현재 메서드 나가기 (Step Out): Shift + F8
- 실행 중인 표현식 평가: Alt + F8
- 브레이크포인트 설정/해제: Ctrl + F8

[버전 관리 (Git)]

- 커밋 창 열기: Ctrl + K
- 푸시: Ctrl + Shift + K
- 변경 사항 보기: Alt + 9
- 변경 사항 비교: Ctrl + D
- 이전 커밋과 비교: Ctrl + Shift + D

[기타 유용한 단축키]

- 액션 찾기: Ctrl + Shift + A
- 실행 구성 실행: Ctrl + Alt + R
- 실행 구성 선택: Alt + Shift + F10
- 디버그 구성 선택: Alt + Shift + F9
- 클립보드 히스토리 보기: Ctrl + Shift + V
- 북마크 설정/이동: F11 / Shift + F11
- 코드 생성 (Getter/Setter 등): Alt + Insert
- 라이브 템플릿 목록 보기: Ctrl + J

===========================================
※ 참고:
- 단축키는 IntelliJ IDEA의 기본 설정 기준입니다.
- 운영 체제나 키맵 설정에 따라 다를 수 있으므로, [Settings > Keymap]에서 확인 및 변경 가능합니다.
- 자주 사용하는 단축키는 Key Promoter X 플러그인을 통해 학습할 수 있습니다.
