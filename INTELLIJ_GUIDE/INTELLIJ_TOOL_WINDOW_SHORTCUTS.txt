


===========================================
 IntelliJ IDEA 툴 윈도우 전환 단축키 정리
===========================================

[Windows / Linux 기준]

┌────────────────────┬────────────────────┐
│ 기능                │ 단축키              │
├────────────────────┼────────────────────┤
│ 프로젝트 창          │ Alt + 1            │
│ 커밋 창             │ Alt + 0            │
│ Git 창             │ Alt + 9            │
│ 서비스 창           │ Alt + 8            │
│ 터미널 창           │ Alt + F12          │
│ 실행 창             │ Alt + 4            │
│ 디버그 창           │ Alt + 5            │
│ 구조 창             │ Alt + 7            │
│ 문제 창             │ Alt + 6            │
│ 북마크 창           │ Alt + 2            │
└────────────────────┴────────────────────┘


[macOS 기준]

┌────────────────────┬──────────────────────┐
│ 기능               │ 단축키                 │
├────────────────────┼──────────────────────┤
│ 프로젝트 창          │ Command + 1          │
│ 커밋 창             │ Command + 0          │
│ Git 창             │ Command + 9          │
│ 서비스 창           │ Command + 8          │
│ 터미널 창           │ Option + F12         │
│ 실행 창             │ Command + 4          │
│ 디버그 창           │ Command + 5          │
│ 구조 창             │ Command + 7          │
│ 문제 창             │ Command + 6          │
│ 북마크 창           │ Command + 2          │
└────────────────────┴──────────────────────┘


[공통 단축키]

┌────────────────────────────────────────┬────────────────────────────┐
│ 기능                                    │ 단축키                      │
├────────────────────────────────────────┼────────────────────────────┤
│ 활성 툴 윈도우 숨기기                     │ Shift + Esc                │
│ 모든 툴 윈도우 숨기기/복원                 │ Ctrl + Shift + F12         │
│ 마지막 활성 툴 윈도우로 이동               │ F12                        │
│ 에디터로 포커스 이동                      │ Esc                        │
└────────────────────────────────────────┴────────────────────────────┘



※ 참고:
- 툴 윈도우 단축키는 IntelliJ IDEA의 기본 설정입니다.
- 사용자 설정에 따라 단축키가 다를 수 있으므로, [Settings > Keymap]에서 확인 및 변경 가능합니다.
