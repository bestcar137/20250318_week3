

===========================================
 IntelliJ IDEA 주석 종류 정리 가이드
===========================================


1. 한 줄 주석 (Single-line comment)
-------------------------------------------
문법:
    // 설명 작성

예시:
    // 이 함수는 합계를 반환합니다.

설명:
    - 간단한 메모용 주석
    - '//' 이후 텍스트는 모두 무시됨


2. 여러 줄 주석 (Multi-line comment)
-------------------------------------------
문법:
    /*
        여러 줄 설명
        주석 처리 가능
    */

예시:
    /*
     여러 줄로 설명 가능
     코드 블럭을 임시로 비활성화할 때 사용
    */

설명:
    - 긴 설명 또는 코드 블럭 전체 비활성화용


3. 문서 주석 (JavaDoc)
-------------------------------------------
문법:
    /**
     * 설명
     * @param ... 매개변수 설명
     * @return 반환값 설명
     */

예시:
    /**
     * 두 수를 더합니다.
     * @param a 첫 번째 수
     * @param b 두 번째 수
     * @return 두 수의 합
     */
    public int add(int a, int b) {
        return a + b;
    }

설명:
    - JavaDoc 문서 생성을 위한 공식 주석
    - IntelliJ에서 자동 완성 및 문서 보기 지원


4. TODO 주석
-------------------------------------------
문법:
    // TODO: 작업 내용

예시:
    // TODO: 예외 처리 추가 필요

설명:
    - 향후 처리해야 할 작업 메모
    - IntelliJ TODO 탭에서 자동 수집됨


5. FIXME 주석
-------------------------------------------
문법:
    // FIXME: 문제 내용

예시:
    // FIXME: NullPointerException 가능성 있음

설명:
    - 문제 코드 표시 및 디버그 가이드
    - TODO와 동일하게 추적 가능


6. @SuppressWarnings (경고 억제)
-------------------------------------------
문법:
    @SuppressWarnings("경고종류")

예시:
    @SuppressWarnings("unchecked")
    List list = new ArrayList();

설명:
    - 컴파일러 경고 억제용 어노테이션
    - 주석처럼 사용되며, 무시하고 싶은 경고에 적용


7. Region 주석 (코드 접기)
-------------------------------------------
문법:
    //region 블럭명
        코드들...
    //endregion

예시:
    //region 초기화 설정
    initConfig();
    //endregion

설명:
    - IntelliJ에서 코드 블럭 접기/펼치기 기능 제공
    - 가독성 향상 및 영역 구분에 유용


8. 언어 하이라이팅 주석
-------------------------------------------
문법:
    //language=언어명

예시:
    //language=JSON
    String json = "{ \"name\": \"ChatGPT\" }";

설명:
    - 문자열 내부에 있는 코드에 하이라이팅 적용
    - JSON, HTML, SQL 등 다양한 언어 지원


기타: 언어별 기본 주석 문법
-------------------------------------------
Kotlin:
    //, /* */, /** */

Python:
    #, """ """

JavaScript:
    //, /* */, /** */
