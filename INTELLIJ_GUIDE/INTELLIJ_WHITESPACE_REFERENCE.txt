

============================================================
📄 공백 문자 종류 및 너비 기준 정리 (복사용) - IntelliJ 참고용
============================================================


[너비 기준 정렬된 공백 문자 목록 + 설명 포함]

[​] ← ZERO WIDTH SPACE (U+200B)
    - 폭 없음 (0em). 시각적으로 보이지 않음
    - 단어 구분, 줄바꿈 유도, 검색 회피 등에 사용

[ ] ← THIN SPACE (U+2009)
    - 아주 좁은 공백 (~0.25em)
    - 숫자와 단위 사이 미세 간격 조정용

[ ] ← NARROW NO-BREAK SPACE (U+202F)
    - 좁고 줄바꿈 방지 (~0.3em)
    - "1 kg"처럼 단위 밀착 표현에 적합

[ ] ← MEDIUM MATHEMATICAL SPACE (U+205F)
    - 중간 폭 공백 (~0.4em)
    - 수식 간격 조절용

[ ] ← EN SPACE (U+2002)
    - 반각 문자 폭 수준 (~0.5em)
    - 단어 간 여백 균형용

[ ] ← 일반 스페이스 (U+0020)
    - 기준값 (1.0em). 키보드 스페이스 바 입력과 동일

[ ] ← EM SPACE (U+2003)
    - 일반 문자 하나만큼 넓은 공백 (1.0em)
    - 더 넓은 시각적 간격이 필요할 때

[　] ← IDEOGRAPHIC SPACE (U+3000)
    - 전각 문자 폭 수준 (2.0em)
    - 한자/일본어 등 전각 언어용 공간 확보


[사용 예시 요약]

- 완전 비가시 분리자 : [​] (U+200B)
  → 텍스트 분리, 검색 회피, URL 구분 등에 사용

- 좁은 간격 필요 시  : [ ], [ ]
  → 숫자 + 단위 ("1 kg", "3 GB") 표현에 적합

- 중간 간격 조절용   : [ ], [ ]
  → 시각 정렬이나 UI/표 균형 보정용

- 넓은 띄어쓰기      : [ ], [ ]
  → 일상적 띄어쓰기나 강조 표현 시 사용

- 전각 문자 정렬용   : [　]
  → 한중일 언어에서 문단 정렬이 필요할 때 사용
